## Sample Research Paper

### Abstract

This paper discusses a simple artificial intelligence workflow for extracting scientific paper information and saving it into Obsidian using Flutter, FastAPI, and Nougat OCR.

### Introduction

The system architecture includes a Flutter desktop application, a Python backend, and an OCR model. The user uploads a PDF file, the backend processes it into markdown format, and the extracted notes are saved into an Obsidian vault. This PDF is intentionally simple and clean so Nougat OCR can process it successfully.
